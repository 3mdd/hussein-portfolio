AIRS is a web-based system developed for iPetro to streamline pressure vessel inspection documentation, photo management, approval workflow, and standardized report generation. The system helps reduce manual work, improve reporting consistency, and support a more efficient inspection process.

Industry Collaboration Web-based System

GitHub: https://github.com/3mdd/AIRS
